## Darflen
<img src="https://static.darflen.com/img/favicons/apple-touch-icon-precomposed.png" align="right" alt="Darflen logo" height="120">
<b>Darflen</b> is a simple yet fully-featured social media website made with PHP where you can share things through posting. Sharing things to many people shouldn't be hard. Darflen makes this easy by removing algorithmic feeds and shares everything into a central hub.

## Disclaimer
<b>Darflen</b> is a personal project made for learning purposes. Do not expect this project to properly work on high-scale production servers. Use it at your own risk as things may break unexpectly. Darf around and find out!
